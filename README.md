# riot-gallery-viewer
Display images in a gallery of thumbnails. View a large version of each by clicking on the thumbnail.

# Glossary
## Gallery
A group of items. Mulitple galleries can be on the same page.
When clicking on the next/previous buttons in the viewer, 

## Gallery Item
  Consists of an image, a (optional) caption, and a clickable html element (usually the image)

## Viewer
The modal that opens open with the web page that displays a large version of the image.
